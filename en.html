Proyectos a Futuro

  • Model of metabolic network using Petri Net

    A metabolic network is the system of biological reactions that govern cell behaviors. Historically, these complete systems have been analyzed with optimization and linear programming. However, given the structure of these systems it makes sense to analyze them discreetly using data structure. This project analyzes metabolic networks modeling them as Petri nets.

  • Testing of Beer using electric nose

    The electric nose is an instruments that through various sensors are able to quantify volatile substances presents in a substances. The idea of this project is to develop a model that managed to automate the tasting of beverages from this instruments.
